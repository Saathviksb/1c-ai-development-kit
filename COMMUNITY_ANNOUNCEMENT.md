# Что это?

**1C AI Development Kit** — это готовая экосистема для работы с AI в 1С разработке. Проект родился из практического опыта использования AI-ассистентов и включает все необходимое для старта.

## Что входит?

### AI-Агенты (12 шт.)

Специализированные агенты для разных задач:
- `onec-code-writer` — написание кода
- `onec-code-reviewer` — код-ревью с проверкой стандартов
- `onec-code-architect` — проектирование архитектуры
- `onec-form-generator` — генерация управляемых форм
- `onec-query-optimizer` — оптимизация запросов
- И еще 7 агентов...

### Навыки (15+ шт.)

Автоматизация рутинных операций:
- Выгрузка/загрузка конфигурации
- Генерация форм, макетов, ролей
- Интеграция с БСП
- Полный цикл разработки функционала

### Правила (9 шт.)

Контроль качества и стандартов:
- Стандарты кодирования для 1С
- Specification-Driven Development
- Управление контекстом и токенами
- Интеграция с BSL Language Server

### MCP-серверы

Описание интеграции с MCP-серверами:
- Бесплатные: BSL LSP Bridge, RLM-toolkit
- Платные: 8 серверов от vibecoding1c.ru

## Quick Start

\`\`\`bash
git clone https://github.com/Arman-Kudaibergenov/1c-ai-development-kit.git
cd 1c-ai-development-kit
cp -r .cursor/* /path/to/your-project/.cursor/
\`\`\`

Перезапустите Cursor и попробуйте:
```
Проверь код модуля CommonModule.ОбщегоНазначения
```

AI автоматически использует агента `onec-code-reviewer` для проверки.
