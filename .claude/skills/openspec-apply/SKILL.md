# /openspec-apply — Реализация изменения

**Guardrails**
- Минимальные реализации — сложность только когда запрошено
- Строго в рамках запрошенного результата
- Читай `openspec/AGENTS.md` для соглашений

**Steps**
1. Прочитай `changes/<id>/proposal.md`, `design.md` (если есть), и `tasks.md` для scope и критериев приёмки
2. Выполняй задачи последовательно, минимальные правки по запрошенному изменению
3. Подтверди завершение каждой задачи перед обновлением статуса
4. Обнови чеклист в `tasks.md`: `- [ ]` → `- [x]` для каждой завершённой задачи
5. После всех задач — финальная проверка

**Reference**
- Используй `openspec show <id>` если нужен дополнительный контекст при реализации
